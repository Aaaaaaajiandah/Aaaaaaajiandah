HepOS

a custom x86_64 exokernel OS, written in Rust from scratch

~/hepos $ ./sysinfo --random 

Overview

HepOS is an exokernel: the kernel only multiplexes hardware (memory, storage, input, display). All OS abstractions — filesystem, windowing, shell — live in a kernel-space libOS for now, with userspace as an ongoing milestone. No Linux, no POSIX, no libc. Bootloader is HepBL, our own from-scratch UEFI bootloader in pure Rust.
